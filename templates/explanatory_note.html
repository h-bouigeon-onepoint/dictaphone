Explanatory Notice

This is an explanatory notice to provide additional information.

Please read and understand the content of this notice carefully.



This program is meant to generate meeting reports automatically from an audio file to increase consultant productivity.

Firstly, we retrieve the transcript of the audio file thanks to whisper API which is deployed on servers hosted by OpenAI.
Then, we use the GPT-3.5-turbo API - deployed on our onepoint private cloud on Azure - to perform the summarization and extract the following information :

  • Participants
  • Agenda
  • Elements Discussed
  • Actions to be taken, Deadlines and Responsible
  • Foot Notes
    • Finally, you can export the result on a Power Point (be aware that some content may be cropped depending on the length of the meeting report).
    Please note that you also have the possibility to change the language in which the summary will be made.
